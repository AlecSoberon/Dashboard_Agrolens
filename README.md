# AgroLens Dashboard v3

Dashboard estático mobile-first para GitHub Pages.

## Archivos

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## Publicación en GitHub Pages

1. Sube estos archivos a la raíz del repositorio.
2. Ve a **Settings → Pages**.
3. Elige **Deploy from a branch**.
4. Selecciona la rama `main` y carpeta `/root`.
5. Guarda y espera a que GitHub genere el enlace.

## Nota

El prototipo usa datos simulados para un fundo de palta Hass. Las alertas son visuales y requieren validación en campo.
